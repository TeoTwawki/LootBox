LootBox

LootBox reads files stored by Final Fantasy XI which contain a local copy of your in game inventory.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to the LootBox homepage.

LootBox is a replacement for the now defunct AltanaCubby. It reads files stored by Final Fantasy XI in the USER folder containing a local copy of your inventory.

The current version can be downloaded here.

Chances are you already have it but to run LootBox you will need the Microsoft Visual C++ 2015 Redistributable Package (x86).

To build LootBox yourself, you will need Microsoft Visual Studio 2019 or later. The free version will work just fine for this. The 2022 edition also works.

Upon first launch, LootBox should automatically detect the installation path of Final Fantasy XI from the registry. If this fails, you will be asked to pick it manually.

If the interface is still empty after all that, open the INI file and verify the FFXiPath value.
The correct path should lead to the directory which contains the USER folder, such as:
C:\Program Files (x86)\PlayOnline\SquareEnix\FINAL FANTASY XI
